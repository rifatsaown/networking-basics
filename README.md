# <div align="center">Networking Basics</div>
### 1. How to find ip of a domain?
```bash
nslookup [domain name] 

#ex : 
nslookup google.com
```
or
```bash
dig [domain name]

#ex :
dig google.com
```
### 2. How to find router ip assigned by the ISP?
```bash
curl ifconfig.me
```
### 3. How to find private ip of host machine?
```bash
ip a
 ```
 or
 ``` 
 ifconfig
 ```
